# lein-plugins

A collection of Leiningen plugins
## Installation 
in project.clj under dev dependencies add 

[lein-plugins "1.0.1"]

The jar i located at clojars.org

## Usage
in projcect.clj 

define :lazytest-path ["src" "test"]

lein lazytest

Or  

lein info
